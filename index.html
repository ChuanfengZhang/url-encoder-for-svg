关于

我们可以在 CSS 中通过 data URI 使用 svg，但是如果不给它编码，只能用在 Webkit 内核的浏览器中。如果用 encodeURIComponent() 编码，就可以在各浏览器中使用了。

SVG 必须有 xmlns 属性，比如：xmlns='http://www.w3.org/2000/svg'。如果没有，这个属性会被自动添加。

编码后的 svg 可用于background 或 border-image。
